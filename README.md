# cam-cross

Откройте терминал и пропишите:<br>
<br>
`pip3 install cv2` <br>
`git clone https://github.com/Firesieht/cam-cross.git`<br>
`cd cam-cross`<br>
`python main.py`<br>
<br>

Далее чтобы запустить можно ответить: да/д/yes/y<br>
Чтобы закрыть нужно нажать ESC, если в консоли то нет/н/no/n<br>
Чтобы сменить камеру можно ответить: с/сменить/сменить камеру/c/change/change cam<br>
